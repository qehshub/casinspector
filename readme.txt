【功能简介】
该软件可批量检验CAS号(化学文摘社号码)，以xlsx格式的excel文件为数据载体，设置简单，易于为大众使用。
该软件的检验包括以下内容：
 - CAS号是否包含非法字符，即是否只由数字和-组成；
 - 如首尾处的非法字符为白字符（如空格、换行符、制表符等），则直接去除；对于位于CAS中间位置的白字符无法去除，需人工处理
 - 检验CAS号最后五位的格式是否符合以下要求：-gh-z，其中ghz为数字
 - 利用CAS号最后一位的校验位，校验CAS号码
软件的输出文件名为“CAS validation log.xlsx”,包含两列，第一列为CAS号码，与待查文件“tbv.xlsx”的CAS号码顺序完全对应，第二列为检验结论，列举并说明如下：
 - 待人工修改（说明：包含软件无法处理的非法字符）
 - 首尾空白字符已清洗且通过校验（说明：处理后的CAS号没有任何问题，您需要修改更新原始数据）
 - 未通过校验！但空白字符已清洗（说明：处理后的CAS号没有非法字符，仅未通过校验，没有其它问题）
 - 校验通过（说明：CAS号没有任何问题）
 - 校验失败！（说明：仅未通过校验，没有其它问题）

【注意事项】
 - 目标文件的文件名必须修改为“tbv.xlsx”
 - “tbv.xlsx”文件的第一列必须为要检验的CAS号
- “tbv.xlsx”文件必须与exe可执行程序位于同一目录下
